# Constitution of India - Educational Platform

An interactive educational platform designed to empower citizens through constitutional awareness. Learn about your rights, duties, and the framework that governs India.

## Features

- **Interactive Learning**: Explore the Constitutional Framework, Fundamental Rights, and Fundamental Duties.
- **Quizzes**: Test your knowledge with modular quizzes.
- **Dashboards**: Separate interfaces for Students, Educators, and Admins.
- **Modern UI**: Built with React, Vite, Tailwind CSS, and Radix UI primitives.

## Component Library

This project uses a custom UI component library built on top of Radix UI for accessible and beautiful interface elements:
- Accordion
- Alert Dialog
- Buttons
- Cards
- Dialogs (Modals)
- And more...

## Tech Stack

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS 4
- **Components**: Radix UI
- **Icons**: Lucide React
- **Animations**: Framer Motion / Motion

## Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
